About CodeSum

Project Overview

Codesum.online is a website I built in a week. The reason for building it so quickly was noticing an influx of developers creating AI tools with real-world utility. I challenged myself to build an AI tool within a week.

Purpose of Codesum

The tool helps people who are new to coding or have never written a line of code get started. Users can enter a prompt and receive three outputs:

  • Key Functions: A list of the key functions used in the code.
  • Explanation: A detailed explanation of what the code does and how each function works.
  • Code: The complete code, ready to copy and paste into an IDE.

Additional Features

I added a help section that guides users on what they need to start coding (e.g., apps, extensions, etc.) and what they need to learn to get a fundemental understanding in the type of code they want to write, whether it be a game, IOS app, AI app etc.

Reflections

Codesum was a fun project, and I am pleasantly surprised by how well it works.

Building it taught me valuable skills in:

  • Time Management: Managing time effectively to meet the one-week deadline.
  • Planning: Exercising efficient planning to achieve the best result.
  • AI Skills: When making this i gained a proficiency in creating and optimising an AI to do specifically what i intended for it to do, very well. I went from a very basic knowledge in AI and to being able to implement & optimise an AI into a website in a single day.
